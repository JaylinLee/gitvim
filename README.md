# gitvim


## First Step

``` bash
curl -fLo ~/.vim/autoload/plug.vim --create-dirs \
    https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim
```

## Then

- git clone https://github.com/JaylinLee/gitvim.git
- cp vimrc ~/.vimrc
- vim .vimrc
- :source ~/.vimrc
- :PlugInstall
- :qa

## Then all things' done.
